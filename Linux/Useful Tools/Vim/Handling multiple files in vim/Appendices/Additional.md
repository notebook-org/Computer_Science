## Additional 
- To open a new file:
```vim
:e <local path to file>
```
- To see all the open files:
```vim
:buffers
```
- To swith between files:
```vim
:b<file number>
```
- To toggle between present and last file:
```vim
ctrl+6
```
- To find a file in a project directory:
```vim
:find <filename or blob pattern>
```
- To Add a new file:
```vim
:edit <directory name>
```
