## Nouns in vim
1. Text Object

Command           | Description
------------------|------------
iw                | Inner word
it                | Inner tag
i"                | Inner quotes
ip                | Inner paragraph
as                | A Sentence
2. Parametrized text object

Command           | Description
------------------|------------
f                 | find the next character
t                 | find the next character
/                 | Search upto next match
